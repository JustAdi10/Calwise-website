# Calwise Website

Static marketing website for the Calwise iOS calorie tracking app.

## Files

- `index.html` - Main landing page
- `privacy-policy.html` - Privacy policy page
- `contact.html` - Contact page
- `styles.css` - Shared styles for all pages

## Deploy To Vercel

1. Push all files in this folder to a GitHub repository.
2. Go to https://vercel.com and click **New Project**.
3. Import the GitHub repository.
4. Keep the default settings (no build configuration needed for this static site) and click **Deploy**.
5. Your site will be live in about 60 seconds.
6. To use a custom domain, open the Vercel project, go to **Settings -> Domains**, and add your domain.

## Notes

- Internal links use relative paths so the site works under any base URL.
- No framework or external CSS library is required.
